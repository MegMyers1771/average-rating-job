# Генератор отчётов по рейтингам ноутбуков

Простая утилита на Python для чтения данных о ноутбуках из CSV-файлов, вычисления среднего рейтинга по брендам и вывода форматированных отчётов в консоль.  
Поддерживается работа с несколькими файлами и автоматическое форматирование таблиц.
